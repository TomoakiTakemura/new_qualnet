Supported RPR FOM files

These files have identical names as in VR-Forces.  In order to run HLA on two 
seperate machines, the same named fed file must exist and be used by both federates.

HLA13 & RPR 1.0 : VR-Link.fed
HLA13 & RPR 2.0017 : VR-Link20017.fed
HLA1516 & RPR 1.0 : VR-Link.xml 
HLA1516 & RPR 2.0017 :VR-Link20017.xml
